Donkey

v1.0.0

Your AI's persistent memory layer.

Stop re-explaining architecture, patterns, and rules in every new chat window. Save context locally, retrieve it instantly.

Capture context automatically

Type @donkey save in any chat. The Cloudflare worker structures your decisions, learnings, and mistakes behind the scenes.

Inject context instantly

Type @donkey use project-x or filter specifically like @donkey avoid mistakes in project-x to load tailored snippets.

100% local and private

All structured memories are saved strictly inside IndexedDB in your browser. No tracking, no external database leaks.
